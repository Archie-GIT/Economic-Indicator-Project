# Economic-Indicator-Project
Developed an end-to-end economic indicators analysis using SQL for data exploration and Microsoft Power BI for interactive visualizations. 
Built dashboards highlighting GDP growth, regional comparisons, and correlations between economic development, internet access, and life expectancy.
